# 创业卡牌游戏 Demo

这是一个以创业流程为核心的卡牌策略网页 Demo，包含自由创业路线和 Mphone 创业案例路线。

## 启动

1. 进入 `游戏demo` 目录。
2. 双击 `启动本地故事Demo.cmd`。
3. 打开 `http://127.0.0.1:4173/index.html`。

正式 Demo 已统一为单一入口。首页、牌库、路线选择、对战、奖励、事件、商店、结算和最终后记均由同一套 `app.js`、`styles.css` 与 `industrial-theme.css` 驱动。

详细入口和维护说明见 [项目文件试玩与 Demo 明细](游戏demo/项目文件试玩与Demo明细.md)，本次整合内容见 [更新说明](游戏demo/更新说明.md)。
