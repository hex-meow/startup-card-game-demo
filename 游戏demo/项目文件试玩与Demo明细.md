# 项目文件、试玩方式与 Demo 明细

## 正式入口

双击 `启动本地故事Demo.cmd`，浏览器访问：

`http://127.0.0.1:4173/index.html`

本地服务不仅提供网页资源，也提供故事进度读写接口。请优先使用启动脚本，不建议直接双击 HTML 文件。

## 文件结构

- `index.html`：唯一正式页面入口。
- `app.js`：游戏数据、页面渲染、流程状态、卡牌效果与交互逻辑。
- `styles.css`：基础布局和通用组件样式。
- `industrial-theme.css`：正式 Demo 的工业像素主题和各阶段页面样式。
- `assets/`：卡牌、人格、章节、事件和首页图片。
- `local_story_server.py`：本地静态服务与故事进度接口。
- `启动本地故事Demo.cmd`：Windows 启动入口。
- `内容编辑器实际Demo表格.xlsx`：内容编辑与参数维护表。

原先独立的首页、路线选择和对战测试 HTML 已合并到 `index.html`，通过参数进入对应页面，避免多个入口的结构和样式不同步。

## 页面直达参数

以下地址均以 `http://127.0.0.1:4173/index.html` 为基础：

| 页面 | 参数示例 |
| --- | --- |
| 正式首页 | 无参数 |
| 初始牌库 | `?view=deck` |
| 路线与关卡选择 | `?view=campaign` |
| 普通对战 | `?view=battle` |
| 章节奖励 | `?ui=demo-reward-persist-v18-20260716&chapter=1` |
| 外部事件 | `?ui=demo-event-persist-v1-20260717&chapter=1` |
| 阶段失败 | `?ui=demo-stage-failure-v1-20260717&chapter=1` |
| 全关卡结局 | `?ui=demo-final-story-v1-20260717` |

调试页参数只用于快速检查界面。正式游玩应从首页开始。

## 当前 Demo 内容

- 自由创业：银发经济、喵工智能、绿喵能源三条路线。
- 创业案例：Mphone 的诞生路线。
- 完整流程：人格选择、初始牌库、关卡选择、对战、奖励、外部事件、商店、出战配置、成功/失败结算和最终后记。
- 资源系统：剩余时间、喵喵币、牌库数量、难度系数和常驻事件。
- 进度保持：奖励、事件和故事进度通过本地服务保存，刷新后可继续当前阶段。

## 维护约定

- 新页面优先并入 `index.html` 与 `app.js`，不要再创建同功能的测试 HTML。
- 通用布局写入 `styles.css`，正式主题覆盖写入 `industrial-theme.css`。
- 运行时图片放入对应的 `assets` 子目录；生成拼图、预览图和日期备份不提交到正式 Demo。
- GitHub 为本项目唯一默认发布远程，不推送到公司 GitLab。
