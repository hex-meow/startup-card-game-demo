# 增加喵喵世界背景与优化 UI

更新日期：2026-07-18

## 更新概述

本次更新为创业卡牌游戏加入更完整的“喵喵世界”视觉背景，并对 Demo 的主要页面、阶段流程和操作体验进行了系统优化。

喵喵世界作为游戏的角色与视觉包装，不会替代原有的创业过程。产品立项、团队组建、资金管理、技术研发、供应链、市场推广和正式发售等核心内容继续使用正常、清晰的创业描述。

## 喵喵世界背景

- 更新首页主视觉，加入喵喵创业团队、工作室和产品研发场景，让游戏世界观更加完整。
- 自由创业路线使用银发经济、喵工智能和绿喵能源三种创业方向，并配置对应的路线插画。
- 加入统一风格的喵群人格图片，用于表现不同成员的能力、性格和创业倾向。
- 更新章节、事件和结局场景，使团队从创意形成、产品开发到最终上市的过程更有连续感。
- 保留 Mphone 创业案例路线，用喵喵世界的视觉风格重新呈现产品诞生过程，同时保留正常的技术、商业和历史事件描述。
- 清理卖鱼、鱼类商品等偏离创业主题的替代文案，避免世界观包装影响玩家理解真实创业流程。

## UI 视觉优化

- 统一首页、喵群选择、初始牌库、路线选择、关卡、对战和结算页面的工业像素视觉风格。
- 重新设计信息层级，放大剩余时间、喵喵币、牌库数量和难度系数等关键参数。
- 减少不必要的顶部阶段栏和重复标题，让关卡内容与主要操作区域更加突出。
- 统一按钮、边框、状态色、章节序号、卡牌窗口和弹窗样式，提高页面辨识度。
- 优化长页面滚动和顶部栏显示，修复滚动后内容重叠的问题。

## 奖励与卡牌页面

- 重做章节奖励页，让三张奖励卡独立展示，强化完成关卡后的奖励感。
- 在奖励页顶部居中显示章节序号和章节名称，并突出当前资源状态。
- 更新自由创业与 Mphone 各章节奖励卡图片，统一采用现代像素插画、黑色边缘和适配卡牌窗口的构图。
- 调整卡牌图片高度和显示比例，减少图片裁切不完整的问题。
- 恢复奖励卡的正常名称与创业含义，避免使用影响理解的临时名称。

## 阶段流程优化

- 重做外部事件页面，仅保留关键资源参数和事件选择，减少无关背景与阶段说明。
- 优化历史事件详情弹窗，使其与其他结算弹窗保持一致的视觉结构。
- 重新设计关卡商店和出战配置页面，并修复完成买卡后无法继续的问题。
- 统一普通关卡成功、阶段失败和全关卡通关结局页面的视觉语言。
- 完善最终后记与后故事汇，加入胜利、完结和继续滚动的视觉提示。
- 调整 Mphone 路线的章节名称、事件名称和时间描述，使页面内容保持自然一致。

## Demo 入口与文件整理

- 将首页、初始牌库、路线选择、对战及各阶段预览整合到统一的 `index.html` 入口。
- 删除重复测试页面和仅用于临时预览的脚本，避免多个页面版本不同步。
- 保留统一的基础样式、正式主题和游戏逻辑文件，方便后续持续维护。
- 保留本地故事进度服务，奖励和事件阶段刷新后仍可继续当前进度。
- 删除旧版图片、AI 生成拼图、预览图、日期备份、生成清单和一次性处理脚本。

## 启动方式

### 方式一：直接打开网页

进入 `游戏demo` 目录，双击 `index.html` 即可打开游戏页面，适合快速查看界面和体验基础玩法。

直接打开时，浏览器地址以 `file://` 开头。受浏览器本地文件安全策略影响，可能出现以下问题：

- 故事进度保存、读取和续写功能无法连接本地服务。
- 刷新或关闭页面后，奖励、事件等阶段状态可能无法正常恢复。
- 部分浏览器可能限制脚本、图片或本地资源加载，导致页面功能不完整。
- 不同浏览器对本地文件的支持方式不同，实际显示效果可能存在差异。

如果直接打开后出现空白页、图片缺失、进度无法保存或按钮无响应，请改用本地服务方式启动。

### 方式二：通过本地服务启动（推荐）

1. 进入 `游戏demo` 目录。
2. 双击 `启动本地故事Demo.cmd`。
3. 浏览器访问 `http://127.0.0.1:4173/index.html`。

本地服务方式可以完整加载页面资源，并支持故事进度保存、阶段状态恢复和故事续写等功能，建议正式试玩时使用此方式。

旧测试页面地址不再保留。需要快速查看指定页面时，可使用 `index.html` 的 `view` 或 `ui` 参数。
