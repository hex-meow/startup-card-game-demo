# 内容编辑器必要参数表

游戏内提供“内容编辑器”入口，可在开局页和路线页打开。编辑器需要覆盖关卡外流程和关卡内对战流程中的关键配置，并能维护 Demo 明细中出现的行业、画像、卡牌、真实案例关卡和历史事件。

| 参数模块 | 必要字段 | 参考示例 | 影响流程 | 说明 |
| --- | --- | --- | --- | --- |
| 路线配置 | 路线ID、路线名称、路线类型、路线简介、最终收官说明 | 模拟创业路线、初代 iPhone 真实故事线 | 关卡外流程 | 路线类型至少包括“模拟创业路线”和“真实公司故事线”，决定开局方式、事件来源和卡牌池。 |
| 模拟创业开局 | 创业者身份、行业方向、产品用户画像、初始时间、时间上限、初始资金、难度系数 | 银发健康、AI效率工具、绿色消费；核心人群、机会人群 | 关卡外流程 | 用于模拟创业路线。身份、行业和画像共同影响初始数值和关卡难度。 |
| 真实案例开局 | 案例名称、历史背景、固定初始时间、固定时间上限、固定初始资金、固定难度系数、初始牌库 | 初代 iPhone、Project Purple | 关卡外流程 | 用于真实公司故事线。玩家选择案例后直接继承确定的初始条件。 |
| 用户画像 | 画像ID、画像名称、画像类型、画像描述、时间修正、资金修正、难度修正 | 核心人群 3 个、机会人群 3 个 | 关卡外流程 | 画像类型至少包括“核心人群”和“机会人群”。 |
| 卡牌参数 | 卡牌ID、卡牌名称、路线归属、卡牌类型、行动力消耗、资金消耗、商店价格、效果类型、效果数值、描述 | 工程师、产品经理、技术预研、天使资金、多点触控原型、Cingular谈判 | 关卡外流程、关卡内对战流程 | 卡牌类型可覆盖人员、技术、资金、管理/策略，以及真实案例专属卡牌。 |
| 牌库配置 | 基础卡牌池、案例专属卡牌池、初始可选数量、初始出战数量、拥有牌库、出战牌库 | 20 张通用创业要素牌中选 8 张；Project Purple 主题牌库 | 关卡外流程、关卡内对战流程 | 模拟线从基础卡牌池选牌；真实线自动带入基础卡牌和案例专属卡牌。 |
| 关卡参数 | 关卡ID、阶段名称、阶段简介、关卡顺序、是否最终关、敌人名称、敌人血条、回合限制、奖励资金、恢复时间 | Project Purple立项、运营商谈判、OS X触控移植、正式发售 | 关卡外流程、关卡内对战流程 | 决定路线地图展示、阶段目标、敌人设置、胜利奖励和最终收官。 |
| 敌人行动 | 行动ID、行动名称、行动类型、意图文本、时间伤害、阻力增加、压力增加、触发回合 | 关键压力、阶段阻力、突发问题、压力上升、最终催办 | 关卡内对战流程 | 敌人每回合按意图行动，可能扣减玩家时间、增加阻力、增加压力或造成高额时间损失。 |
| 奖励参数 | 奖励ID、所属关卡、奖励选项数量、奖励卡牌、奖励资金、恢复时间、奖励描述 | 阶段奖励 3 选 1 | 关卡外流程 | 阶段胜利后触发，通常从 3 张奖励牌中选择 1 张加入拥有牌库。 |
| 事件参数 | 事件ID、事件名称、事件类型、触发路线、触发阶段、历史日期、正面效果、负面效果、常驻影响、描述 | 政策补贴、供应短缺、Cisco商标诉讼、FCC批准、WWDC Web 2.0应用 | 关卡外流程、关卡内对战流程 | 事件类型包括标准时间线事件、随机意外事件和历史时间线事件。 |
| 商店参数 | 商店ID、可售卡牌范围、基础价格、价格修正、已拥有过滤规则、刷新规则 | 购买尚未拥有的基础卡；事件降低卡牌价格 | 关卡外流程 | 关卡之间进入商店，玩家花费资金购买新卡。 |
| 重组规则 | 初始出战上限、每关上限增长、最低出战张数、最高出战张数、是否允许重复卡 | 至少 8 张；每完成一关出战上限 +1 | 关卡外流程 | 控制进入下一关前的出战牌库调整。 |
| 存储与重置 | 保存状态、本地存储键、重置默认内容、是否即时刷新当前页面 | 保存后下次打开仍然生效；重置恢复默认内容 | 编辑器流程 | 修改应立即影响当前页面；保存后写入浏览器本地存储。 |

卡牌效果逻辑目前以固定规则为主，编辑器优先维护展示信息和关键数值；后续如果开放效果脚本或条件逻辑，可以在卡牌参数中增加效果触发条件、目标范围和组合效果。
